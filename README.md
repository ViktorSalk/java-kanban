# java-kanban

### Распределение функциональности между классами:

***Task:*** Базовый класс для всех типов задач, содержащий общие свойства и методы.
***Subtask:*** Подкласс Task, представляющий подзадачи, связанные с эпиками.
***Epic:*** Подкласс Task, представляющий эпики, которые могут содержать подзадачи.
***TaskManager:*** Менеджер задач, отвечающий за управление всеми задачами, эпиками и подзадачами.


### Взаимодействие классов:

***TaskManager*** создает и управляет экземплярами ***Task***, ***Epic*** и ***Subtask***.
***TaskManager*** предоставляет методы для добавления, обновления, удаления и получения задач, эпиков и подзадач.
***Epic*** содержит список ***Subtask*** и обновляет свой статус на основе статусов подзадач.
***Subtask*** ссылается на ***Epic***, к которому он принадлежит.


### Наследование:

***Subtask*** наследует от ***Task***, что позволяет ему использовать общие свойства и методы.
***Epic*** наследует от ***Task***, что позволяет ему использовать общие свойства и методы, а также добавлять функциональность для управления подзадачами.


### Хранение задач:

***TaskManager*** использует ***HashMap*** для хранения задач, эпиков и подзадач, что обеспечивает быстрый доступ по идентификатору.


### Интерфейс:

Приложение не имеет явного интерфейса, но ***TaskManager*** предоставляет методы для взаимодействия с задачами, эпиками и подзадачами.
